# Альтернативы туннелей для Telegram WebApp (РФ)

## Проблема
Многие популярные сервисы (ngrok, cloudflared) заблокированы в РФ. Нужен работающий туннель с HTTPS.

## Рабочие решения для РФ

### 1. xTunnel (рекомендуется) ✅

**Сайт:** https://xtunnel.ru

**Преимущества:**
- Работает в РФ без VPN
- Поддерживает HTTPS
- Бесплатный тариф доступен
- Простая настройка

**Как использовать:**
1. Зарегистрируйтесь на https://xtunnel.ru
2. Создайте новый туннель
3. Укажите локальный порт: `5173`
4. Получите HTTPS URL
5. Используйте этот URL в BotFather и `.env`

### 2. Tuna (если доступен)

**Сайт:** https://tuna.ru

**Как использовать:**
1. Зарегистрируйтесь
2. Создайте туннель на порт 5173
3. Получите HTTPS URL

### 3. Serveo (через SSH)

**Как использовать:**
```bash
ssh -R 80:localhost:5173 serveo.net
```

**Преимущества:**
- Не требует регистрации
- Работает через SSH
- Бесплатно

**Недостатки:**
- Может быть нестабильным
- URL меняется при каждом запуске

### 4. LocalXpose (lpx)

**Установка:**
```bash
# Скачайте с https://localxpose.io
# Или через npm (если доступен)
npm install -g localxpose
```

**Использование:**
```bash
lpx http 5173
```

### 5. Serveo альтернатива (через веб)

Можно использовать веб-версию Serveo, если доступна.

## Рекомендация

**Для стабильной работы используйте xTunnel:**
- ✅ Работает в РФ
- ✅ Стабильный
- ✅ HTTPS поддержка
- ✅ Простая настройка

## Настройка после получения URL

1. **BotFather:**
   - `/mybots` → ваш бот → "Bot Settings" → "Menu Button"
   - URL: ваш HTTPS URL от туннеля

2. **`.env` файл:**
   ```env
   telegram_webapp_url=https://ваш-url.xtunnel.ru
   ```

3. **Перезапустите приложения**

## Troubleshooting

### Туннель не работает
- Проверьте, что фронтенд запущен на порту 5173
- Убедитесь, что туннель активен
- Проверьте логи туннеля

### Telegram не открывает WebApp
- Убедитесь, что URL начинается с `https://`
- Проверьте, что URL доступен из браузера
- Убедитесь, что URL в BotFather и `.env` совпадают

### Страница-предупреждение
- Некоторые туннели показывают предупреждения
- Telegram не может их обойти
- Используйте туннели без предупреждений (xTunnel, Tuna)
